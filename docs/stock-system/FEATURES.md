# 功能说明

## 1. 信号监控

### 1.1 每日信号汇总
- 自动计算200只股票的信号状态
- 按评分排序推荐
- 显示买入/卖出信号

**信号计算逻辑**：
- 买入信号：前7天平均分 ≥ 30分
- 卖出信号：涨幅≥20%（止盈）或跌幅≥8%（止损）
- 冷却期：卖出后1天内不再买入同一只股票

### 1.2 信号推送
- 每日19:30推送信号到QQ
- 推送内容：买入信号、卖出信号、持仓状态

**推送格式**：
```
📊 2026-06-12 信号报告

🟢 买入信号（3只）：
• 贵州茅台 (600519) 评分: 45.2
• 宁德时代 (300750) 评分: 42.8
• 比亚迪 (002594) 评分: 41.5

🔴 卖出信号（1只）：
• 中国平安 (601318) 止盈 +22.5%

📈 当前持仓（2只）：
• 隆基绿能 (601012) 浮盈 +5.2%
• 五粮液 (000858) 浮亏 -2.1%

💰 模拟收益：+12.35%
```
