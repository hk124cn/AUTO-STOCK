# 测试用例

## 1. 信号监控测试

### TC-001: 每日信号计算
- **输入**：200只股票的评分数据
- **预期**：正确计算7日平均分和信号状态
- **验证**：
  - 信号数量正确
  - 信号类型正确（买入/卖出）
  - 评分计算正确

### TC-002: 信号排序
- **输入**：10只股票的信号
- **预期**：按评分从高到低排序
- **验证**：
  - 排序正确
  - 推荐标记正确（⭐）

### TC-003: 信号推送
- **输入**：今日信号数据
- **预期**：正确推送到QQ
- **验证**：
  - 推送内容正确
  - 推送时间正确（19:30）
