## 1. 财务因子 (20分)

**文件**: `src/factors/financial_factor.py`
**权重**: 20分（满分）

### 1.1 指标体系

| 指标 | 权重 | 正增长满分 | 负增长封顶 |
|------|------|-----------|-----------|
| 扣非净利润同比增长率 | 50% | 10分 | -5分 |
| 归母净利润同比增长率 | 25% | 5分 | -2.5分 |
| 营业总收入同比增长率 | 25% | 5分 | -2.5分 |

### 1.2 基础分计算

**正增长**：`基础分 = 满分 × min(同比增长率 / 40, 1)`

- 40%增长得满分，不封顶
- 线性插值，40%以上不再增长

**负增长**：`原始分 = 封顶分 × (rate / 100)`，`基础分 = 原始分 / 10`

- 例如扣非增长率 -30%：50 × (-30/100) / 10 = -1.5分

### 1.3 趋势评分

基于最近3个季度的同比增长率变化趋势：

| 趋势类型 | 加分值 | 说明 |
|----------|--------|------|
| 连续增长 | +7.5% × 满分 | Q-2 < Q-1 < Q |
| 连续下降 | -7.5% × 满分 | Q-2 > Q-1 > Q |
| V型反转 | +5% × 满分 | 负→正转 |
| 倒V反转 | -5% × 满分 | 正→负转 |
| 整体上升 | +5% × 满分 | 平均变化 > 5% |
| 整体下降 | -5% × 满分 | 平均变化 < -5% |

趋势评分范围：±10% × 满分

### 1.4 总分计算

```
最终评分 = 基础分总和 + 趋势分总和
范围限制：[-10, 20]
```

### 1.5 示例

| 股票 | 报告期 | 扣非增长率 | 归母增长率 | 营收增长率 | 得分 |
|------|--------|-----------|-----------|-----------|------|
| 茅台600519 | 2025Q4 | -30.83% | -30.34% | -19.35% | -4.29分 |
| 福耀玻璃600660 | 2026Q1 | -17.32% | -15.68% | +5.08% | -1.88分 |

## 2. 单日涨跌幅因子 (10分)

**文件**: `src/factors/daily_change_factor.py`
**权重**: 10分（满分）

### 2.1 评分逻辑

**第一步：趋势判断**

基于5日/20日均线关系：
- `strong_up`: 短期均线 > 长期均线 5% 以上
- `weak_up`: 0% < 差值 ≤ 5%
- `weak_down`: -5% ≤ 差值 ≤ 0%
- `strong_down`: 差值 < -5%

**第二步：评分矩阵**

根据趋势状态，采用不同评分矩阵：

| 涨跌幅区间 | strong_up | weak_up | weak_down | strong_down |
|-----------|-----------|---------|-----------|-------------|
| (-10%, -7%] | 8 | 7 | 6 | 9 |
| (-7%, -3%] | 6 | 5 | 4 | 7 |
| (-3%, 0%] | 4 | 3 | 2 | 3 |
| (0%, 3%] | 2 | 4 | 5 | 6 |
| (3%, 7%] | 1 | 6 | 7 | 8 |
| (7%, 10%] | 0 | 3 | 8 | 9 |

**第三步：成交量修正**

当成交量放大（量比 > 1.5）时：
- 下跌趋势中股价上涨（反弹）：×1.2
- 上涨趋势中股价下跌（回调）：×1.2

**特殊规则**：
- 涨停（>9.5%）且处于下跌趋势：10分
- 跌停（<-9.5%）且处于上涨趋势：1分

### 2.2 返回数据

```python
{
    "name": "单日涨跌幅",
    "score": 8.5,  # 0-10
    "sum_score": 10,
    "meta": {
        "today_change": 3.25,      # 当日涨跌幅%
        "trend_status": "weak_up",  # 趋势状态
        "volume_ratio": 1.8         # 量比
    }
}
```

## 3. 5日涨跌幅因子 (10分)

**文件**: `src/factors/fiveday_factor.py`
**权重**: 10分（满分）

### 3.1 评分规则

基于最近5个交易日（含今日）的收益率：

| 5日收益率 | 得分 |
|---------|------|
| > 10% | 10分 |
| 5% ~ 10% | 8分 |
| 0% ~ 5% | 6分 |
| -5% ~ 0% | 4分 |
| < -5% | 2分 |

### 3.2 计算方法

```
5日收益率 = (今日收盘价 / 5日前收盘价 - 1) × 100%
```

## 6. 关注度因子 (10分)

**文件**: `src/factors/attention_factor.py`
**权重**: 10分（满分）

### 6.1 评分公式

```python
基础分 = (平均关注指数 - 80) / 2 - 标准差 / 4 + 市场涨跌幅 / 10
基础分 = clamp(基础分, 0, 7)
```

### 6.2 稳定奖励

当 平均关注指数 < 85 且 标准差 < 3 时：
```
稳定奖励 = 2.5 - (标准差 / 3)
```

### 6.3 最终得分

```
最终得分 = clamp(基础分 + 稳定奖励, 1, 10)
```

### 6.4 数据说明

- 使用最近20个交易日关注度数据
- 市场涨跌幅用于辅助调整

| 新闻数量 | 得分 |

| 增速 | 得分 |

## 附录

### A. 因子基类接口

所有因子继承 `src/core/base_factor.py` 的 `BaseFactor`：

```python
class BaseFactor:
    def __init__(self, code, name=None):
        self.code = code

    def calculate(self) -> dict:
        """返回因子评分结果"""
        raise NotImplementedError
```

### B. 返回值格式

```python
{
    "name": "因子名称",
    "score": 8.5,       # 得分（已标准化到sum_score）
    "sum_score": 10,     # 满分（用于标准化）
    "detail": {...},     # 可选，明细数据
    "meta": {...}        # 可选，元数据
}
```

### C. 添加新因子

1. 在 `src/factors/` 下创建新文件，如 `my_factor.py`
2. 定义类继承 `BaseFactor`
3. 实现 `calculate()` 方法
4. 系统会自动发现并加载

```python
from src.core.base_factor import BaseFactor

class MyFactor(BaseFactor):
    def calculate(self):
        # 评分逻辑
        return {
            "name": "我的因子",
            "score": 8.0,
            "sum_score": 10
        }
```
