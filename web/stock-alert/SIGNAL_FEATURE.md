# 买入信号功能说明

## 功能概述

在个股评分预警图中添加买入信号标记，帮助用户快速识别买入机会。

## 实现原理

### 信号计算

- **买入信号条件**: 前7天平均分 ≥ 30分
- **计算公式**: `avg7 = sum(过去7天评分) / 7`
- **信号标记**: 评分曲线上显示绿色圆点

### 数据来源

- **评分数据**: `result/score_price_history.csv`
- **价格数据**: `data/price/{code}.csv`

## 使用方法

1. 访问 https://auto-claw.top/yujing/
2. 搜索股票（如: 600519）
3. 开启"评分曲线"和"买入信号"
4. 评分曲线上会显示绿色圆点标记买入信号
5. 鼠标悬停在圆点上会显示信号详情

## 信号解读

- **绿色圆点**: 前7天平均分 ≥ 30分，买入信号
- **无标记**: 前7天平均分 < 30分，观望

## 策略参数

根据回测结果，推荐参数：

- **买入阈值**: 30分（前7天平均分）
- **止盈**: 20%
- **止损**: 8%
- **冷却期**: 1天

## 注意事项

1. 信号是基于历史评分计算的，不构成投资建议
2. 需要结合其他因素（如市场环境、个股基本面）综合判断
3. 回测结果不代表未来收益
