# 通用审查清单

每个模块过一遍：

## 正确性
- [ ] 边界（空数据集、单元素、最后一天）
- [ ] NaN/None/0 不会引发崩溃
- [ ] 时间序：point-in-time？前视偏差？
- [ ] 缓存：过期策略、键冲突、并发安全
- [ ] 错误处理：是否吞噬异常？

## 性能
- [ ] 大数据下（5000 股 × 5 年）能否 5 分钟内跑完
- [ ] 重复 IO 是否会被缓存命中
- [ ] 是否有 O(n²) 算法

## 可维护性
- [ ] 命名是否一致
- [ ] 函数是否过长（>200 行？）
- [ ] 魔法数字是否抽出
- [ ] 死代码

## 部署
- [ ] 硬编码路径
- [ ] 时区
- [ ] 中文路径/编码
- [ ] 跨平台（Linux vs Mac）

## 安全（仅 deploy/api/）
- [ ] 任何地方 hardcoded token/key
- [ ] API 是否有认证
- [ ] CORS
- [ ] SQL 注入（虽然 SQLite，但要看）

## 文档
- [ ] CLAUDE.md 与实际是否一致
- [ ] HANDOVER.md 是否过期
- [ ] README 在新目录下存在
